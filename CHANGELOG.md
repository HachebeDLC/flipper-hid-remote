# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-02
### Added
- Initial Release.
- Full Keyboard USB HID Bridge with multi-layout support.
- Precision Mouse/Trackpad with non-linear acceleration.
- Bluetooth Name customization and Battery reporting.
- PWA Web Dashboard for Android, iOS, and PC.
- SD Card persistence for application settings.
